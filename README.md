# P1_Laberinto
Programa que localiza los caminos de un laberinto. Optimizado para uso en GNU/Linux.

Primer proyecto para la materia de Algoritmos y Estructura de Datos de la Facultad de Ingeniera de la Universidad Nacional Autónoma de México.

Compilar con gcc.

Para ejecutar, escribir en consola el nombre del ejecutable seguido de el nombre del archivo que contiene el laberinto, seguido de las coordenadas (en formato de arreglo bidimensional) del punto donde quiera iniciar.
